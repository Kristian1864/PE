Step 4 — Manager remuneration in €:
Mock Q1 example (compute carry %) — CC = 400, Final = 2,250, Hurdle = 8%, Input carry rate = 30%
Step 1: Global IRR = (2,250 − 400) ÷ 400 = 462.50%  →  Step 2: Net Global IRR = 462.50% − 8% = 454.50%  →  Step 3: Carry % = 454.50% × 30% = 136.35% ✓
Trap: 138.75% = forgetting to subtract the hurdle (462.50%×30%). 462.50% and 454.50% are the other two options — correct intermediate values, wrong stopping points.

Mock Q7 example (compute €remuneration) — Global IRR = 348%, Carry % = 108% (given), Final = 1,750
Step 1: CC = 1,750 ÷ (1 + 3.48) = 1,750 ÷ 4.48 = 390.625M  →  Step 4: Remuneration = 108% × 390.625 = €421.875M ✓
Note: the 8% hurdle is already baked into the given 108% carry figure — do NOT subtract it again.

Enterprise Value → Equity Value → PE Proceeds EXAM SAMPLE TYPE

EV = 24,000  →  Equity = 20,000  →  Proceeds = 6,000  →  IRR = (6,000 ÷ 4,500)^(1/3) − 1 ≈ 10.06%

ROE Leverage Formula LBO CORE

If ROI > i → every unit of leverage increases ROE. If ROI < i → every unit of leverage destroys value. The spread (ROI − i) is the key variable.

Leverage Multiple → Buying Power EXAM SAMPLE TYPE

Example: LP equity = $500M, leverage = 3×  →  Buying power = $500M × 3 = $1.5 billion

Italian PEX — Participation Exemption TAX

Effective rate ≈ 1.4%. Condition: holding period >1 year, no real estate. Example: gain = €200M → taxable base = €10M → tax = €2.79M (per Prof. Zanchi’s clarification, use the Session 8 Excel-file rate of 27.9%, not the 27.4% shown in some slides)

Weighted Average Anti-Dilution — New Conversion Price VC CONTRACTS

Example: 1M shares @ €10, issue 500K new @ €6
New CP = (1M × 10 + 500K × 6) ÷ 1.5M = 13,000,000 ÷ 1,500,000 = €8.67
Full ratchet (simpler): New CP = new issue price directly. No formula needed.
